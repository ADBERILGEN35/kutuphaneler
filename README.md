# kutuphaneler
Kurgulanan kütüphaneler veri tabanı 	MySQL 8 teknolojisi ile hayata geçirilmiştir. MySQL ile oluşturduğumuz veri tabanımıza Spring Boot Kotlin yardımı ile Swagger UI oluşturularak veri ekleme, silme , güncelleme gibi işlemlerin gerçekleştirilmesi daha stabil ve kolay bir hale getirilmiştir. Localhost üzerinden Swagger UI erişimi gerçekleştirilerek kütüphane, kitap, yazar, kategori gibi alanlara veri ekleme, silme ve güncelleme yapılırken. Emanet girişi yapılan kitaplar için sql sorguda left join kullanarak Emanet verilen kitabın adı, kitabın alındığı kütüphanenin adı, üyenin adı-soyadı, emanet alınma tarihi ve teslim edilme tarihi gibi verilerin tek sorguda gösterilmesi sağlanmıştır. Veri tabanı işlemleri Spring Boot’ta jdbc template üzerinden direkt bir db connection alarak raw sql sayesinde controllerlar ile yapılmıştır. Kısıtlı yönetici arayüzü için React kullanılmıştır.
